Quizz App is an interactive web-based application designed to test users' knowledge throught timed multiple-choice questions.
The system allows users to answer questions, receive instant feedback and view their final score at the end of each quiz session.
It is built with a simple and user-friendly interface to ensure smooth navigation and accessibility.
This project demonstrates core programming concepts such as data handling, conditional logic, user interaction and basic application flow, making it suitable for educational purpose and learning-focused environments.
